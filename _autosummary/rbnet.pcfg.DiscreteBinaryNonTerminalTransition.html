The output of this function is typically handled by a custom implementation of Cell.inside_mixture().

Parameters:

  • location – location of the variable for which to compute the inside marginals

  • inside_chart – a lookup chart with inside probabilities for other variables

  • terminal_chart – a lookup chart with values of the terminal variables

  • value – value of the variable for which to compute the inside marginals or None; if value is provided this means that conditional inside probabilities should be computed (for the given value); otherwise general inside marginals (as a function of the variable) should be computed; also see RBN.non_terminals()

Returns:

array-like or iterable with inside probabilities; if the transition is not possible, this should be of length zero or contain only inside probabilities representing zero
